# Deadline Outbreak (PC Zombie FPS)

A browser-based **first person shooter** where you battle waves of zombies across **10 increasingly difficult levels**, ending with a **final boss fight**.

## Features
- FPS-style perspective combat with mouse aim + click/space to fire
- 10 levels of rising difficulty (more enemies, higher enemy stats)
- Final boss on level 10 with bigger health pool and heavier damage
- Health, ammo, score, and remaining-zombie HUD
- Sound effects (shots, hits, growls, victory fanfare)
- Dynamic synthesized background music with mute toggle

## Controls
- **Mouse move**: Aim
- **Click / Space**: Shoot
- **R**: Reload
- **M**: Toggle music
- **Pause button**: Pause/unpause game

## Run locally
```bash
python3 -m http.server 8000
```
Then open `http://localhost:8000`.
